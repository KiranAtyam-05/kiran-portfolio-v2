# Kiran Atyam Portfolio V2

Advanced interactive style variant with magnetic buttons, spotlight cursor, scroll progress, section reveal animations, theme modes, voice hover, animated avatar, and resume-driven content.

Run locally from this folder:

```powershell
python -m http.server 4174 --bind 127.0.0.1
```

Update portfolio data by replacing `public/resume/resume.txt`.
